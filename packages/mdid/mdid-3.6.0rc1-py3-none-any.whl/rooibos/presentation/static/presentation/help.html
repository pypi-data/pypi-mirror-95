Viewer Keyboard Shortcuts

  • Move to next slide
  • Move to previous slide
  • Space Select next viewer
  • Shift Space Select previous viewer
  • s Sync/unsync viewers
  • r Reset viewer
  • h Hide/show image
  • i Show/hide image metadata
  • t Show/hide image title
  • Zoom in
  • Zoom out
  • u Unsplit viewer
  • y Split viewer vertically
  • x Split viewer horizontally
  • f Enter/exit full screen mode

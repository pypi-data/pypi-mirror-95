# cli guide
<b>init</b>, <b>serve</b> are available
```powershell
# calling cli
pyvuejs_cli [command]
# or
python -m pyvuejs_cli [command]

# init
pyvuejs_cli init [project_name(default="./")]

# serve
# must run in project directory
pyvuejs_cli serve
```

# Ape Accounts Plugin

Plugin for [Ape Ethereum Framework](https://github.com/ApeWorX/ape) for working with local accounts

## Dependencies

* [python3](https://www.python.org/downloads) version 3.6 or greater, python3-dev

## Installation

### via `pip`

You can install the latest release via [`pip`](https://pypi.org/project/pip/):

```bash
pip install ape-accounts
```

### via `setuptools`

You can clone the repository and use [`setuptools`](https://github.com/pypa/setuptools) for the most up-to-date version:

```bash
git clone https://github.com/ApeWorX/ape-accounts.git
cd ape-accounts
python3 setup.py install
```
